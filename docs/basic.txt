-- Estructura basica de la app.

La app esta pensada en manager de jugadores para la posibilidad de transferencias y analisis de jugadores y torneos.
Se implementa CRUD con json como metodo de tranferencia de datos atraves de Rest.

Para consultar el modo de instalacion y las tecnologias utilizadas leer el README.md

-OBS:
La app puede sufrir modificaciones.
